88 /*!
89  * \ingroup CoordinateMapsGroup
90  * \brief Spatial rotation in three dimensions using Euler angles
91  *
92  * Rotation angles should be specified in degrees.
93  * First rotation \f$\alpha\f$ is about z axis.
94  * Second rotation \f$\beta\f$ is about rotated y axis.
95  * Third rotation \f$\gamma\f$ is about rotated z axis.
96  * These rotations are of the \f$(\xi,\eta,\zeta)\f$ coordinate system with
97  * respect
98  * to the grid coordinates \f$(x,y,z)\f$.
99  *
100  * The formula for the mapping is:
101  * \f{eqnarray*}
102  * x &=& \xi (\cos\gamma \cos\beta \cos\alpha - \sin\gamma \sin\alpha)
103  * + \eta (-\sin\gamma \cos\beta \cos\alpha - \cos\gamma \sin\alpha)
104  * + \zeta \sin\beta \cos\alpha \\
105  * y &=& \xi (\cos\gamma \cos\beta \sin\alpha + \sin\gamma \cos\alpha)
106  * + \eta (-\sin\gamma \cos\beta \sin\alpha + \cos\gamma \cos\alpha)
107  * + \zeta \sin\beta \sin\alpha \\
108  * z &=& -\xi \cos\gamma \sin\beta + \eta \sin\gamma \sin\beta
109  * + \zeta \cos\beta
110  * \f}
111  */
