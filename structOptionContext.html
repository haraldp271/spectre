Detailed Description

Information about the nested operations being performed by the parser, for use in printing errors. A default-constructed OptionContext is printed as an empty string. This struct is primarily used as an argument to PARSE_ERROR for reporting input file parsing errors. Users outside of the core option parsing code should not need to manipulate the contents.
