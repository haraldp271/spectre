Detailed Description

template<typename System>
struct Actions::ComputeBoundaryFlux< System >

Receive boundary data from neighbors and compute boundary contribution to the time derivative.

Note
The data must be oriented by the sender, but this class handles projection, etc.

Uses:

  • ConstGlobalCache: Metavariables::numerical_flux
  • DataBox: Tags::Element<volume_dim>, Tags::HistoryBoundaryVariables<Direction<volume_dim>, system::variables_tag>, Tags::Extents<volume_dim>, Tags::TimeId, Tags::UnnormalizedGridNormal<volume_dim>

DataBox changes:

  • Adds: system::dt_variables_tag
  • Removes: Tags::HistoryBoundaryVariables<Direction<volume_dim>, system::variables_tag>
  • Modifies: nothing
