Detailed Description

template<typename Receiver>
struct Actions::SendDataForFluxes< Receiver >

Compute local boundary data needed for fluxes and send it to neighbors.

Note
The data is oriented for the receiver, but the receiver is responsible for projection, etc.

Uses:

  • ConstGlobalCache: Receiver
  • DataBox: system::variables_tag, Tags::Element<volume_dim>, Tags::Extents<volume_dim>, Tags::TimeId

DataBox changes:

  • Adds: Tags::HistoryBoundaryVariables<Direction<volume_dim>, system::variables_tag>
  • Removes: nothing
  • Modifies: nothing
Template Parameters
Receiverthe parallel_component to send data to
