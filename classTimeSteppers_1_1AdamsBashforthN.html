An explanation of the computation being performed by this function:

We number the sides \(1\) through \(S\), and let the evaluation times on side \(s\) be \(\ldots, t^s_{-1}, t^s_0, t^s_1, \ldots\), with the starting location of the numbering arbitrary. The union of all these sets of times produces the sequence \(\ldots, \tilde{t}_{-1}, \tilde{t}_0, \tilde{t}_1, \ldots\), again numbered arbitrarily. For example, one possible sequence of times for a two-sided boundary is:

\begin{equation} \begin{aligned} \text{Side 1:} \\ \text{Union times:} \\ \text{Side 2:} \end{aligned} \cdots \begin{gathered} t^1_4 \\ \tilde{t}_8 \\ t^2_1 \end{gathered} \leftarrow \Delta \tilde{t}_8 \rightarrow \begin{gathered} \, \\ \tilde{t}_9 \\ t^2_2 \end{gathered} \leftarrow \Delta \tilde{t}_9 \rightarrow \begin{gathered} t^1_5 \\ \tilde{t}_{10} \\ \, \end{gathered} \cdots \end{equation}

We call the indices of the start and end times in the union time set \(n_S\) and \(n_E\), respectively, so for the above example, if we wish to compute the step on side 1 from \(t^1_4\) to \(t^1_5\), we would have \(n_S = 8\) and \(n_E = 10\). The indices of steps on each side containing or starting with union time number \(n\) are denoted \(\vec{m}_n\), so in the above example \(\vec{m}_9 = (4, 2)\). The union indices corresponding to the start of a set of side step indices \(\vec{m}\) are denoted \(\vec{n}_{\vec{m}}\), so in the example \(\vec{n}_{(4, 2)} = (8, 9)\). In the below equations, addition, subtraction, min, and max operations applied to vectors operate on each component individually, i.e., \(\vec{q} + k = (q^1 + k, \ldots, q^S + k)\).

Let \(k\) be the order of the integrator. For a sequence of times \(t^s_{j_s - (k-1)}, \ldots, t^s_{j_s}\) on each side \(s\), we can write the cardinal functions on those times as

\begin{equation} \newcommand\of[1]{\!\!\left(#1\right)} C_{a_1, \ldots, a_S}(t_1, \ldots, t_S; j_1, \ldots, j_S) = \prod_{s=1}^S \ell_{a_s}\of{t_s; t^s_{j_s - (k-1)}, \ldots, t^s_{j_s}}, \end{equation}

with \(\ell_a(t; x_1, \ldots, x_k)\) the Lagrange interpolating polynomials. (Note that we take the indices \(a\) to run from \(1\) to \(k\) here, but from \(0\) to \(k-1\) in the code.) The cardinal function satisfies \( C_{a_1, \ldots, a_S}(t^1_{i_1}, \ldots, t^S_{i_S}; j_1, \ldots, j_S) = \prod_{s=1}^S \delta_{i_s, j_s + a_s - k} \). We write the integral of a cardinal function using an Adams-Bashforth step based on the union times along the diagonal \(t_1 = t_2 = \cdots = t_S\) from \(\tilde{t}_n\) to \(\tilde{t}_{n+1}\) as

\begin{equation} \newcommand\of[1]{\!\!\left(#1\right)} I_{n,\vec{q}} = \Delta \tilde{t}_n \sum_{j = 0}^{k-1} \tilde{\alpha}_{nj} C_{\vec{q}-\vec{m}_n+k} \of{\tilde{t}_{n - j}, \ldots, \tilde{t}_{n - j}; \vec{m}_n}, \end{equation}

where \(\tilde{\alpha}_{nj}\) are the Adams-Bashforth coefficients derived from the union times and \(\vec{q}\) is a vector specifying the cardinal function indices. The boundary delta for the step is then

\begin{equation} \newcommand\mat{\mathbf} \mat{F}_{n_S, n_E} = \mspace{-7mu} \sum_{\vec{q} = \vec{m}_{n_s}-(k-1)}^{\vec{m}_{n_E-1}} \mspace{-4mu} \mat{D}_{\vec{q}} \mspace{-4mu} \sum_{n = \max\left\{n_S, \vec{n}_{\vec{q}} \right\}} ^{\min\left\{n_E, \vec{n}_{\vec{q} + k}\right\} - 1} \mspace{-7mu} I_{n,\vec{q}}, \end{equation}

where \(\mat{D}_{\vec{q}}\) is the coupling function evaluated at the side values at side step indices \(\vec{q}\).
