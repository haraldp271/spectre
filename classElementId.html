Detailed Description

template<size_t VolumeDim>
class ElementId< VolumeDim >

An ElementId uniquely labels a Element. It is constructed from the BlockId of the Block to which the Element belongs and the VolumeDim SegementIds that label the segments of the Block that the Element covers.
