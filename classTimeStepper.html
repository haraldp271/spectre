const std::vector< std::deque< std::tuple< Time, BoundaryVars, FluxVars >>> &  history,
const TimeDelta &  time_step 
) const
inlinenoexcept

Compute the change in a boundary quantity due to the coupling on the interface.

The history is a vector representing the different sides of the interface, with the local side first. Each entry in that vector is the time history of a side, similar to the argument to update_u.

The coupling function coupling should compute the derivative of the boundary quantity from the side data, supplied as a std::vector<std::reference_wrapper<const FluxVars>>. These values may be used to form a linear combination internally, so BoundaryVars should have appropriate mathematical operators defined to allow that.

References Parallel::value.
