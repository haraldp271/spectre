◆ apply_visitor()

template<typename Invokable , typename... Variants, typename... Args>
void apply_visitor ( boost::variant< Variants... > &  box,
Args &&...  args 
)

Calls an Invokable's apply static member function with the current type in the boost::variant.

The primary use case for this is to allow executing a single Action at any point in the Algorithm. The current best-known use case for this is setting up initial data. However, the implementation is generic enough to handle a call at any time that is valid. Here valid is defined as the apply function only accesses members of the DataBox that are guaranteed to be present when it is invoked, and returns a DataBox of a type that does not break the Algorithm.
